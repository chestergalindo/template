# Template - Frontend

Template with Vite, React, postcss and typescript.

### 🚀 Enlaces
 * **First Version:** [page](https://templatevite.netlify.app)

#### 🛠 Installation of project
```
npm install
```

### 🖥 Execution
```
npm run dev
```

This project start in port http://localhost:3000

### 🛠️ Build with

The tools used
  * Frontend: React
  * css: postcss
  * typescript
